---
name: Module-by-module workflow
description: User wants to work one module at a time, say "next" to proceed, get commit messages after each module
type: feedback
---

Work one module at a time. Wait for user to say "next" before proceeding to the next module. After completing each module, provide a commit message for the user to push to GitHub. Keep project structure simple. Frontend should use React.
**Why:** User wants granular control over progress and wants to commit/push after each milestone.
**How to apply:** Complete module fully, suggest commit message, then stop and wait.
